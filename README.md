# NAC2026 Seminar Dashboard
Dashboard สำหรับติดตามข้อมูลการจัดสัมมนาในงาน **NSTDA Annual Conference 2026 (NAC2026)**

## เกี่ยวกับโปรเจกต์
หน้านี้ถูกสร้างขึ้นเพื่อสรุปข้อมูลสำคัญ (Executive Dashboard) สำหรับผู้บริหารและทีมงาน NSTDA เพื่อดูความคืบหน้าของหัวข้อสัมมนา วิทยากร และสถานะต่าง ๆ ในภาพรวม

## ลิงก์เข้าชม Dashboard
คุณสามารถดู Dashboard ในรูปแบบเว็บไซต์ได้ที่นี่:
[คลิกเพื่อเปิด Dashboard](https://pjpreme-ui.github.io/nac2026_seminar_dashboard/)

## โครงสร้างไฟล์
* `index.html`: ไฟล์หลักที่ใช้แสดงผล Dashboard
* `README.md`: รายละเอียดโปรเจกต์ (ที่คุณกำลังอ่านอยู่)
